# twarc_jsonl-to-txt

This folder contains simple Python-scripts that write tweets from jsonl files into a simple txt file for easy text analysis.
The twarc tools for collecting twitter data produce jsonl files. However, for analysis the data with the Corpus analysis tool AntConc I needed simple txt files. 

# Links
twarc: https://scholarslab.github.io/learn-twarc/
AntConc: https://www.laurenceanthony.net/software/antconc/
